Atributos

Proporcionan informacion adicional sobre los elementos HTML

  • Atributos href

    • Especifica la URL de la pagina a la que va el enlace

  • Atributo a

    • Define un hipervinculo

  • Atributo src

    • Especifica la ruta a la imagen que se mostrara

  • Atributos ancho y largo

    • La etiqueta img debe contener los atributos width y height que especifican ancho y largo de la imagen

  • Atributo Alt

    • Especifica un texto alternativo para una imagen si por algun moivo la imagen no se puede mostrar

    *alt (Imagen de un atardecer en la playa)

  • Atributo de Idioma

    • Siempre deb incluir el "lang" dentro de la etiqueta html para declarar el idioma de la pagina web

  • Atributo Title

    • Define informacion adicional sobre un elemento

Etiqueta aside

Representa una seccion de una pagina que consite
en contenido que esta indirectamente relacionada
con el contenido principal.
Algo que esta anexo al contenido

Etiqueta header

Representa un grupo de ayudas introductorias o de navegacion.
Puede contener algunos elementos de encabezados
asi como tambien un logo, un formulario de busqueda
un nombre de autor y otros componentes.

Etiqueta Section

Define y ayuda para separar secciones en un documento

Etiqueta Footer

Contiene informacion general del autor o
empresa, enlaces a documentos o el famoso copyright.

Etiqueta main

Especifica el contenido principal de el documento.
Solo puede contener un main en toda la pagina.
Va dentro de la etiqueta body.

Etiqueta span

Funciona para agrupar elementos y darles estilo.

Metadatos HTML

  1. Metadato Title

    2. Define el titulo de la página que se muestra
    en la barra del titulo del navegador.

    Se utiliza para los motores de busqueda

  2. Metadato Charset utf-8

    3. Especifica la codificacion de caracteres
    utilizados en la pagina.

  3. Metadato Description

    4. Proporciona una descripcion concisa
    de la pagina que puede ser utilzada por
    los motores de busqueda.

  4. Metadato Keywords

    5. Especifica palabras clave relevantes para la página

  5. Metadato viewport

    6. Ajusta el ancho de la página del ancho del dispositivo
    y establece la escala inicial.
